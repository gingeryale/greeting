Welcome to the homepage of my site, built in Jekyll - Static Site Generator
(when there is no database, there is nothing to hack)

I am a graphic designer / UX/UI/ product designer & web developer. I have designed interfaces for online products at startups, as well as real life products such as packaging and bottles.

I've spent years learning graphic design at SUNY - the State University of New York, where I got a degree in Communication & Graphic Design. Taking night classes allowed me to work at various positions, honing my skills in the graphic arts as well as the web.

This site is built in part using CSS Grid, with Flexbox as a fallback for older browsers. This website uses SASS, which came baked-into the Jekyll static site generator.

Why Jekyll over Wordpress?

Jekyll is lightweight. Is supported by Github Pages. Jekyll is a simple commit message away in git, and Github Pages will quickly update it. It's free to host on Github.
